Architecture deployed

5.1 VPC · Subnets (AZ1 + AZ2) · NACLs · Security Groups · NAT Gateway Internet Gateway · Route Tables · VPC Endpoints (S3, DynamoDB, SecretsManager) 5.2 Route 53 private zone (cloudbank.internal) ALB (bank-alb) · 6 Lambda target groups · path-based listener rules 5.3 Cognito User Pool · MFA (TOTP) · App Client · Hosted Domain 5.4 Amplify app (cloud-bank-frontend) 5.5 Lambdas: auth / accounts / transactions / notifications / kyc Per-Lambda IAM roles · X-Ray tracing · VPC placement · DLQ 5.6 Secrets Manager (MongoDB + RDS) · S3 (audit, KYC, monitoring) RDS PostgreSQL · DynamoDB (sessions, audit-events, accounts, kyc-status) ElastiCache Redis 5.7 KMS (master key + alias) · WAF v2 (Common + SQLi rules) GuardDuty · ACM Certificate 5.8 SNS (alerts + transaction-events) · SQS (queue + DLQ) CloudWatch alarms (Lambda errors, DLQ depth, RDS CPU) CloudTrail · AWS Backup (daily @ 02:00 UTC, 30-day retention)
